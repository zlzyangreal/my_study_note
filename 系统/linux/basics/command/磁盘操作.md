## 分区与挂载
### 查看设备挂载情况
lsblk 或者  lsblk -f
* -f 详细信息
### 磁盘情况查询
**查询系统整体磁盘使用情况**
* df -h
**查询指定目录的磁盘占用情况，默认当前目录**
* du （选项）（目录）
选项：
* -s                  指定目录占大小总汇
* -h                  带计量单位
* -a                  含文件
* -max-depth=[深度]   子目录深度
* -c                  列出明细的同时，增加汇总值
## 磁盘实用指令
### 统计/opt文件夹下文件的个数
* ls -l /opt |grep "^-"| wc -l
~~~
    - 表示普通文件
    wc 用于统计行数
    -l 统计输入行的数量
~~~

### 统计/opt文件夹下的目录的个数
* ls -l /opt |grep "^d"| wc -l
### 统计/opt文件夹文件的个数，包括子文件夹里的
* ls -lR /opt |grep "^-"| wc -l
### 统计/opt文件夹目录的个数，包括子文件夹里的
* ls -lR /opt |grep "^d"| wc -l
